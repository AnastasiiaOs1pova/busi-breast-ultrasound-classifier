# Классификатор УЗИ-снимков груди (BUSI)

Репозиторий содержит проект по классификации УЗИ-снимков молочной железы на основе датасета BUSI с использованием transfer learning.

В проекте реализованы:
- классификация изображений по классам;
- интерпретация предсказаний с помощью Grad-CAM;
- baseline-подход к поиску аномалий с использованием autoencoder.

## О проекте

Задача проекта — построить модель для анализа УЗИ-снимков груди и сравнить два подхода:

1. **Supervised classification** — классификация изображений с помощью предобученной CNN-модели;
2. **Anomaly detection baseline** — автоэнкодер для базового поиска аномальных изображений.

Проект выполнен на медицинских изображениях и ориентирован на построение воспроизводимого ML pipeline: от подготовки данных до интерпретации результатов.

В рамках проекта я:

- подготовила и предобработала датасет BUSI;
- реализовала pipeline обучения модели с использованием transfer learning;
- обучила классификатор для распознавания классов на УЗИ-снимках;
- провела оценку качества модели по основным метрикам;
- интерпретировала предсказания модели с помощью Grad-CAM;
- построила baseline autoencoder для поиска аномалий;
- сравнила результаты классификационного и reconstruction-based подходов.

## Датасет

В проекте используется датасет **BUSI (Breast Ultrasound Images Dataset)**.

**Важно:** изображения не включены в репозиторий. Датасет находится по ссылке: https://www.kaggle.com/datasets/sabahesaraki/breast-ultrasound-images-dataset
